# Product Overview

家計簿Webアプリ。Laravel + MySQL で構築され、個人利用の家計管理（収支・予算・定期支出・分割払い・キャッシュフロー予定）を一括で扱う。

## Core Capabilities

- **取引明細**: 収入/支出の登録・編集・削除、年月・種別での絞り込み
- **予算管理**: カテゴリ別予算の設定と進捗確認
- **定期支出**: ルール定義と月初の自動展開（Artisan コマンド）
- **分割払い**: 分割プランと支払い記録
- **キャッシュフロー予定表**: 予定の一覧・同期・編集
- **データ入出力**: CSV インポート/エクスポート（Maatwebsite/Excel）

## Target Use Cases

- 日々の収支を取引明細で記録し、予算と照らし合わせる
- 定期支出をルールで定義し、月初に一括で取引に展開する
- 分割払いをプラン管理し、支払い予定と実績を把握する
- キャッシュフロー予定表で将来の入出金を確認する
- 他システムとの連携やバックアップのため CSV で入出力する

## Value Proposition

- 認証済みユーザー単位でデータを完全分離（マルチテナント的運用）
- ゼロダウンタイムデプロイとロールバック可能な本番運用
- さくらインターネット等のレンタルサーバー向けデプロイ手順を用意
